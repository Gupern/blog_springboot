# blog_springboot
a blog by vue(frontend) and springboot(backend), this is the springboot part


# 约定
- dao：数据库访问的object
- dto：接收前端的object
- vo：返回给前端的view object
- pojo：其他无法区分的object

# 目录结构
- 按官网建议，按模块来分，而不是按功能来分，更方便开发，详见: `https://docs.spring.io/spring-boot/docs/2.4.4/reference/htmlsingle/#using-boot`
